## 📊 주요 기능 상세

### 1. 창업 통합 정보 (`/info`)
**5개 탭으로 구성**:
- 창업 절차: 8단계 체계적 가이드
- 초기 비용: 4개 업종별 상세 분석
- 지원금: 6개 주요 지원금 정보
- 프랜차이즈: 4개 카테고리별 브랜드
- 창업 팁: 6가지 성공 노하우

**특징**:
- 실용적이고 구체적인 정보
- 체크리스트 형태
- 시각적 강조 (아이콘, 색상)

### 2. 상권 분석 (`/analysis`)
**분석 항목**:
- 유동인구: 일평균, 연령별, 성별, 시간대별
- 경쟁 현황: 점포 수, 밀집도, 분포
- 매출 정보: 평균 매출, 성장률, 비교
- 종합 평가: 적합도 점수 (0-100), 등급 (A-D)

**시각화**:
- 막대 차트: 연령대별 유동인구
- 라인 차트: 시간대별 추이
- 도넛 차트: 업종 분포
- 지도: 위치 표시 (준비 완료)

**데이터**:
- 현재: 데모 데이터 (실제 같은 샘플)
- 향후: 소상공인 API 연동 (코드 준비 완료)

### 3. AI 컨설팅 (`/consulting`)
**입력 정보**:
- 예산: 6단계 선택
- 지역: 자유 입력
- 업종: 자유 입력
- 경험: 5단계 선택
- 목표: 자유 입력

**분석 결과**:
- 창업 적합도: 점수 + 등급 + 코멘트
- 추천 아이템: 3가지 (이름, 이유, 비용)
- 예산 분석: 카테고리별 상세 계획
- 지원금 안내: 받을 수 있는 지원금
- 핵심 조언: AI 맞춤형 조언

**AI 채팅**:
- OpenAI GPT-4 활용
- 실시간 대화형 인터페이스
- 맥락 기반 답변
- 키워드 기반 데모 응답 (API 키 없을 시)
